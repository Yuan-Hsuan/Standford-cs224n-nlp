## Study Tips

1. **Watch & Read First**: Read the recommended papers and watch the lecture video before attempting the assignment.
2. **Take Notes**: Create a `.md` note file in this folder for each lecture to write down formulas, derivations (especially backpropagation gradients), and conceptual architectures.
3. **Use PyTorch Documentation**: Keep the PyTorch API documentation open when implementing models from Lecture 5 onwards.
