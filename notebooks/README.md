# Notebooks Sandbox

Use this directory to create, run, and save Jupyter notebooks for exploring NLP concepts, libraries, and datasets.

## Suggested Exploration Notebooks
1. **`01_word2vec_visualization.ipynb`**: Visualizing pre-trained embeddings (Word2Vec, GloVe, FastText) using t-SNE or PCA.
2. **`02_pytorch_basics.ipynb`**: PyTorch tensor operations, automatic differentiation (autograd), and building a simple neural network.
3. **`03_huggingface_pipelines.ipynb`**: Experimenting with zero-shot classification, summarization, and translation using HuggingFace `transformers`.
4. **`04_tokenization_exploration.ipynb`**: Visualizing byte-pair encoding (BPE) and WordPiece tokenization outputs.
